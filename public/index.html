About This API

This API allows you to generate responses based on prompts using Googleâ€™s Generative AI. You can save, retrieve, update, and delete generated responses easily.

API Endpoints

  • POST /api/generate: Generate and store a response.
  • GET /api/responses: Retrieve all stored responses.
  • GET /api/responses/:id: Retrieve a specific response by ID.
  • PUT /api/responses/:id: Update a specific response by ID.
  • DELETE /api/responses/:id: Delete a specific response by ID.

Get Started

To generate a response, send a POST request to /api/generate with your prompt in the body.
